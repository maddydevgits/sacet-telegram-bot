# About Project
SACET Telegram Bot is a chatbot which would return the student details whenever they pass the roll number to the chatbot.

# Softwares
1. VS Code
2. Python3

# Packages
1. pip install -r requirements.txt

# Execution Process
1. Install softwares
2. Clone the Repo
3. Install packages
4. Python app.py
5. Open the Telegram Bot: @sacetbot
